ABOUT US

Jiha Dance Studio is a dance studio that teaches various dance genres ranging from traditional Indonesian dance, hip hop, breakdancing, modern dance, ladies style, cover dance KPop, contemporary and others. Each genre will be taught by professional teachers who are experienced in their fields.


BEGINNER'S CLASS

Beginner’s class is a class for beginners who are just starting to learn dance. This class will teach you the basics of dance moves for each type of dance


STANDARD'S CLASS

Standard's class is a class for dancers who already understand the field of dance to be followed. This class will sharpen your understanding of dance techniques in each field.


HIGH'S CLASS

High's class is a class for dancers who are very skilled and ready to be given a high level of technique.
